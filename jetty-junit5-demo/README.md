# jetty-junit5-demo

A simple example that explains how to do the tests with Jetty server and Bootique.

## Prerequisites

* Java 17 or newer
* Apache Maven

## Test the Demo

```bash
git clone git@github.com:bootique-examples/bootique-jetty-demo.git
cd bootique-jetty-demo/jetty-junit5-demo
mvn clean test
```
